Three-Story Corporate Network Infrastructure

This network topology is designed for a three-story company structure. Each floor contains a switch, a router, and an access point. Inter-floor connections are established through three routers with routing enabled. Each department is isolated using a different VLAN. Wireless access is provided on each floor via access points. The network includes a total of 8 VLANs, 3 routers, 3 switches, and various client devices.
